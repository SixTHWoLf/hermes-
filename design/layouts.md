# Layouts - 页面布局设计

## 整体布局

```
┌────────────────────────────────────────────────────────────────┐
│ Header (64px)                                                   │
│ ┌──────────────────────────────────────────────────────────┐   │
│ │ Logo + 标题          [备份 ▼] [恢复] [设置]              │   │
│ └──────────────────────────────────────────────────────────┘   │
├──────────┬─────────────────────────────────────────────────────┤
│ Sidebar  │ Main Content                                        │
│ (200px)  │                                                      │
│          │ ┌────────────────────────────────────────────────┐   │
│ ◉ Model  │ │ 页面标题 + 描述                                 │   │
│ ○ Agent  │ ├────────────────────────────────────────────────┤   │
│ ○ Terminal│ │                                                 │   │

```

## 响应式断点

| 断点 | 宽度 | 布局变化 |
|------|------|----------|
| Mobile | < 640px | 侧边栏折叠为汉堡菜单 |
| Tablet | 640px - 1024px | 侧边栏 180px |
| Desktop | > 1024px | 完整侧边栏 200px |
