# Hermes Console UI 设计规范

## 概述

Hermes Console 是一个用于配置和管理 Hermes Agent 的 Web 控制台，采用深色主题、简洁现代的设计风格。

## 设计原则

1. **深色主题优先** - 减少视觉疲劳，适合长时间使用
2. **简洁现代** - 去除不必要的装饰，突出功能
3. **中文界面** - 全中文文案，本地化体验
4. **组件高复用** - 基于 shadcn/ui 模式，确保一致性

## 技术栈

- **框架**: Next.js + TypeScript
- **样式**: Tailwind CSS
- **组件库**: shadcn/ui
- **图标**: Lucide React

## 模块结构

### MVP 范围（3个核心模块）

| 模块 | 路径 | 描述 |
|------|------|------|
| Model 配置 | `/model` | 模型选择、Provider 配置、API Key 管理 |
| Agent 配置 | `/agent` | 对话参数、人格管理、推理设置 |
| Terminal 配置 | `/terminal` | Docker 配置、容器资源、超时设置 |

### 后续模块（优先级排序）

- Browser 配置
- Auxiliary 配置
- Display 配置
- TTS/STT 配置
- Security 配置
- 消息平台集成
- Code Execution 配置
- Memory 配置
